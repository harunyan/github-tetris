# GitHub Tetris

簡易テトリス（React + Canvas）

## 実行方法

1. 依存をインストール

```
npm install
```

2. 開発サーバー起動

```
npm run dev
```

ブラウザで http://localhost:5173 を開くと動作します（Vite のデフォルトポート）。

## 操作
- ← / → : 左右移動
- ↓ : ソフトドロップ
- Space : ハードドロップ
- Z / X : 回転
- C : ホールド
